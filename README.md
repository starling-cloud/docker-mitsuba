# Docker | Mitsuba

**Starling Studio Suite**

---

## Installation

- **Build the Docker image**:

  ```bash
  docker build -t mitsuba-renderer .
  ```

- **Run a container using your new image**:

  ```bash
  docker run -it mitsuba-renderer
  ```
